Contributors: sloped
Tags: admin,styles
Requires at least: 3.2
Tested up to: 3.2
Stable tag: 0.5

Updates the font-face of the html editor back to those used in 3.1.
== Description ==

Adds new stylesheet to the admin interface to return to the font family used in 3.1 in the HTML editor. 

== Installation ==


1. Upload `font-face-admin.zip` to the `/wp-content/plugins/` directory
2. unzip font-face-admin.zip
3. Activate the plugin through the 'Plugins' menu in WordPress

Done.
